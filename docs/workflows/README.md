# Workflows

This directory documents key workflows (development, CI/CD, releases, production deployment, versioning).

## Files
- `local-development.md` — setup/commands for local work.
- `ci-cd.md` — pipeline steps and gates.
- `releases.md` — release process (to be expanded).
- `versioning.md` — versioning policy (to be expanded).
- `production-deployment.md` — production deploy process (to be expanded).
- Compliance/DSR considerations are woven into CI/CD, releases, and production deployment checklists; keep them in sync as features land.
- `release-preflight.md` — pre-release checklist template.
- `postmortem-template.md` — lightweight postmortem template.
- `pr-checklist.md` — checklist to use in PRs/reviews to ensure docs/tests/compliance coverage.
